# Third-Party Runtime Components

Local Whisper Studio downloads, but does not redistribute, the following components:

- FFmpeg Windows builds from BtbN/FFmpeg-Builds. FFmpeg is licensed under the GNU GPL/LGPL depending on build configuration.
- whisper.cpp from ggml-org/whisper.cpp, licensed under the MIT License.
- Whisper model weights hosted by ggerganov/whisper.cpp on Hugging Face. Review the upstream model license and terms before redistribution.

The application installer contains none of these runtime binaries or model files.
