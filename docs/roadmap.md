# Roadmap

1. Establish repository documentation and architecture decisions.
2. Define the client-owned local meeting model.
3. Build an authenticated ASR gateway with ephemeral storage and cleanup.
4. Add user-provided remote credentials to Electron.
5. Validate live transport for English meetings.
6. Implement English live transcription around speaker-turn dialogue.
7. Consider Vietnamese batch transcription after the v1 path is stable.

Vietnamese realtime transcription and durable server-side meeting storage are explicitly out of
scope.
